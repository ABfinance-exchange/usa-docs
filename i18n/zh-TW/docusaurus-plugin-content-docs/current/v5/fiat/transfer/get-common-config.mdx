批量獲取業務配置。傳入 `scene=abf_order_config` 可取得訂單列表的篩選枚舉(`paymentType` 與 `status`)。

### HTTP請求
GET `/fiat/public/channel/common-config-batch`

### 請求參數
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|scene |**true** |string |業務配置場景。傳入 `abf_order_config`。多值以英文逗號分隔。 |

### 響應參數

| Parameter | Type | Comments|
|:----- |:-----|------- |
|paymentType |array |支付類型選項 |
|> name |string |選項名稱 |
|> ckey |string |i18n key |
|> value |string |選項值 |
|status |array |狀態選項 |
|> name |string |選項名稱 |
|> ckey |string |i18n key |
|> value |string |選項值 |

### 請求示例

```http
GET /fiat/public/channel/common-config-batch?scene=abf_order_config HTTP/1.1
Host: api-testnet.abfinance.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-RECV-WINDOW: 5000
```
