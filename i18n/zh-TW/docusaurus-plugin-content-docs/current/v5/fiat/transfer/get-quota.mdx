查詢用戶當前 KYC / VIP 等級以及下一等級的日 / 週 / 月 / 年累計交易額度。轉帳出款流程復用渠道提現額度配置。

:::tip
- ABF 轉帳出款流程傳入 `bizCode=PAY_BANK`、`transCode=PAY_BANK_PAY_OUT`。
- 舊值 `WITHDRAW` / `WITHDRAW_APPLY` 等仍可用於跨流程額度查詢。
:::

### HTTP請求
GET `/fiat/public/channel/level-quota`

### 請求參數
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fiat |**true** |string |法幣幣種，目前固定為 `USD` |
|bizCode |**true** |string |產品代碼。轉帳出款使用 `PAY_BANK`。其他值（`DEPOSIT`、`WITHDRAW`、`EXPRESS`）分別對應各流程 |
|transCode |**true** |string |交易代碼。轉帳出款使用 `PAY_BANK_PAY_OUT`。其他值（`DEPOSIT_APPLY`、`WITHDRAW_APPLY`、`EXPRESS_APPLY`）分別對應各流程 |

### 請求示例

```http
GET /fiat/public/channel/level-quota?fiat=USD&bizCode=PAY_BANK&transCode=PAY_BANK_PAY_OUT HTTP/1.1
Host: api-testnet.abfinance.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-RECV-WINDOW: 5000
```
