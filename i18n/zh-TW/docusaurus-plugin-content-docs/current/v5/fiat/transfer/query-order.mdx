依訂單編號查詢單筆 ABF 轉帳訂單，包含收款人資訊、金額拆分、各幣種兌換明細及申訴資訊（若觸發風控）。

### HTTP請求
GET `/fiat/private/channel/abf/transfer/order`

### 請求參數
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderNo |**true** |string |訂單編號 |

### 請求示例

```http
GET /fiat/private/channel/abf/transfer/order?orderNo=xxx HTTP/1.1
Host: api-testnet.abfinance.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-RECV-WINDOW: 5000
```
