Fetch business configuration in batch. Pass `scene=abf_order_config` to retrieve the order-list filter enums (`paymentType` and `status`).

### HTTP Request
GET `/fiat/public/channel/common-config-batch`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|scene |**true** |string |Business configuration scene. Pass `abf_order_config`. Multiple values are comma-separated. |

### Response Parameters

| Parameter | Type | Comments|
|:----- |:-----|------- |
|paymentType |array |Payment-type options |
|> name |string |Option name |
|> ckey |string |i18n key |
|> value |string |Option value |
|status |array |Status options |
|> name |string |Option name |
|> ckey |string |i18n key |
|> value |string |Option value |

### Request Example

```http
GET /fiat/public/channel/common-config-batch?scene=abf_order_config HTTP/1.1
Host: api-testnet.abfinance.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-RECV-WINDOW: 5000
```
