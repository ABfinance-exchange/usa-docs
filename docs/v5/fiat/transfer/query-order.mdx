Query a single ABF transfer order by order number, including payee details, amount breakdown, per-currency conversion details, and appeal information (if triggered).

### HTTP Request
GET `/fiat/private/channel/abf/transfer/order`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|orderNo |**true** |string |Order number |

### Request Example

```http
GET /fiat/private/channel/abf/transfer/order?orderNo=xxx HTTP/1.1
Host: api-testnet.abfinance.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: xxxxxxxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-RECV-WINDOW: 5000
```
