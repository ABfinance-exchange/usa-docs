Retrieve all bank accounts associated with the authenticated user, returned in ascending order by creation time.

:::tip
- Results are returned in ascending order by creation time.
:::

### HTTP Request
GET `/v5/fiat/bank-accounts`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |**true** |string |Currency code to filter bank accounts. |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|------- |
|retCode |integer |Return code. 0 indicates success. |
|retMsg |string |Return message. |
|result |object |Result object. |
|> data |array |List of bank account objects. |
|>> bankName |string |Bank name. |
|>> bankAccountId |string |Bank account ID. |
|>> accountNumberMasked |string |Masked bank account number. |
|>> status |string |Account status: pending_verification &#124; active &#124; rejected. |
|>> createdAt |integer |Account creation time in milliseconds timestamp. |

### Request Example

```http
GET /v5/fiat/bank-accounts?currency=USD HTTP/1.1
Host: api.bybit.com
X-BAPI-API-KEY: YOUR_API_KEY
X-BAPI-TIMESTAMP: 1717804800000
X-BAPI-SIGN: YOUR_SIGNATURE
```
