Subscribe to real-time status changes of ABF fiat cashier orders (e.g., ABF transfer orders created via [Create Transfer Order](../../fiat/transfer/create)).

**Topic:** `fiat.cashier.order`

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|topic |string |Topic name, `fiat.cashier.order` |
|data |object |Payload |
|> orderNo |string |Transaction order number |
|> [orderStatus](../../enum#fiattransferorderstatus) |string |Order status |
|> bizCode |string |Product code |
|> transCode |string |Transaction code |

### Subscribe Example

```json
{
    "op": "subscribe",
    "args": [
        "fiat.cashier.order"
    ]
}
```

### Stream Example

```json
{
    "topic": "fiat.cashier.order",
    "data": {
        "orderNo": "",
        "orderStatus": "1005",
        "bizCode": "PAY_BANK",
        "transCode": "PAY_BANK_PAY_OUT"
    }
}
```
